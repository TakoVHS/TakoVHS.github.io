# CogniPrint Website (GitHub Pages)

Public website repository for **CogniPrint** at `https://cogniprint.org`.

## Deployment note
- Deployment is performed by `.github/workflows/pages.yml` on pushes to `main`.
- GitHub Pages serves the repository root.
- `CNAME` must contain `cogniprint.org`.
- `.nojekyll` is present to disable Jekyll processing.
- Paper landing page is `/papers/`; manuscript PDF path is `/papers/cogniprint-v0.1.pdf`.
