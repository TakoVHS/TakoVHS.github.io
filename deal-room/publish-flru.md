# FL.ru Submission Pack

## Profile headline

AI-first technical operator: аудит IT-проектов, Telegram-автоматизация, HR-воронки

## Profile description

Помогаю предпринимателям и малым B2B-командам приводить IT-проекты и операционные процессы к понятному рабочему состоянию.

Основные направления:

1. Аудит и стабилизация MVP-репозиториев.
2. Telegram-автоматизация заявок и внутренних процессов.
3. HR Autopilot для массового подбора.

Работаю через ограниченный результат: диагностика, список блокеров, план исправлений, воронка, документация и передача.

## Service 1

### Title

Аудит MVP-репозитория перед запуском

### Description

Если у вас есть проект в GitHub, но он нестабилен, плохо документирован или не готов к демонстрации, проведу техническую диагностику и дам понятный план действий.

Результат:

- отчёт о текущем состоянии;
- список блокеров;
- карта рисков;
- рекомендации по запуску;
- план следующего спринта.

## Service 2

### Title

Telegram-бот и воронка для массового подбора персонала

### Description

Собираю Telegram-first воронку для подбора курьеров, швей, операторов и линейного персонала.

Что можно настроить:

- анкета кандидата;
- первичный скрининг;
- статусы;
- передача рекрутеру;
- напоминания;
- отчёт по этапам.

## Service 3

### Title

Telegram-автоматизация заявок и рабочих процессов

### Description

Помогаю перевести ручные Telegram-процессы в управляемую систему: заявки, статусы, уведомления, админский flow и отчёты.

## First response

Здравствуйте. Пришлите кратко текущую задачу, что уже есть, что сейчас не работает и какой результат нужен. После этого предложу оптимальный формат: диагностика, пилот или спринт.
