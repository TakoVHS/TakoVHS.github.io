# AI Deal Room — Sales Pack

This folder contains public-facing sales assets for two commercial offers:

1. **AI Repo Rescue Sprint** — recovery and launch readiness for chaotic AI-built or contractor-built repositories.
2. **HR Autopilot** — Telegram-first automation for mass hiring funnels.

The materials are designed for immediate outbound sales: one-pagers, commercial proposal structures, intake forms, sample report templates, objection handling and first-message scripts.

## Primary offer to sell first

**AI Repo Rescue Diagnostic**

- Entry product: $500–$1,500
- Timeline: 2–3 days
- Output: technical audit, launch blockers, risk map and rescue sprint estimate
- Upsell: $4,000–$7,000 Rescue Sprint

## Secondary offer

**HR Autopilot Pilot**

- Entry product: 150,000 RUB
- Timeline: 14 days
- Output: Telegram candidate funnel, screening form, statuses, recruiter handoff and funnel report
- Upsell: 300,000–500,000 RUB implementation + monthly support

## Sales rule

Do not sell vague development work. Sell bounded outcomes with evidence: audit, blockers, sprint scope, funnel map, smoke checks, docs and handover.
