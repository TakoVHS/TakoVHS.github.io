# Proposal Templates

## 1. AI Repo Rescue Diagnostic

### Title

AI Repo Rescue Diagnostic

### Goal

Create a clear technical state report for an existing repository and define the shortest safe path to demo or launch.

### Scope

- repository structure review;
- dependency and run-path review;
- documentation review;
- deployment-path review;
- launch blocker map;
- rescue recommendation.

### Deliverables

- technical diagnostic report;
- blocker list;
- risk map;
- launch-readiness recommendation;
- estimate for rescue sprint.

### Timeline

2–3 working days.

### Price

$500–$1,500 depending on project size and urgency.

### Next step after diagnostic

If the diagnostic confirms a practical rescue path, the client can continue into a 7-day or 14-day sprint.
