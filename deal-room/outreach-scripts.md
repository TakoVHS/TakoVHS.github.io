# Outreach Scripts

## AI Repo Rescue — first message

Hi. I help founders and small teams whose product already has a repository, but the project is currently hard to run, hard to deploy or hard to verify.

My format is an AI Repo Rescue Sprint: audit, launch blockers, smoke checks, deployment notes, documentation and a clear PR plan.

The entry point is a paid Repo Diagnostic: 2–3 days, technical report, blocker map and rescue estimate. If useful, I can show the report structure.

## AI Repo Rescue — short version

Hi. If your AI-built MVP has code but is not ready for demo or launch, I can run a 2–3 day Repo Diagnostic: technical audit, launch blockers and a practical rescue plan.

## AI Repo Rescue — follow-up 1

Quick follow-up. The diagnostic is useful when the team already has code but lacks clarity: what works, what is broken, what blocks launch and what should be fixed first.

It is not a vague development retainer. It is a short technical state report with next steps.

## AI Repo Rescue — follow-up 2

If timing is not right for a full rescue sprint, the diagnostic can still work as a standalone deliverable. You can use the report with your internal team or decide later whether to continue.

The goal is not to replace recruiters. The goal is to reduce losses between a candidate response and an actual shift start.

I can show a 14-day pilot structure for one vacancy and one city.

Hello. If you hire couriers or other high-volume staff, I can set up a 14-day Telegram hiring funnel pilot: screening form, statuses, recruiter handoff and conversion report.

## HR Autopilot — follow-up 1

## HR Autopilot — follow-up 2

We can start small: one vacancy, one city, one funnel. No CRM migration is required at the pilot stage.

# Objection handling

## We already have developers

Great. I do not replace your team. I provide an independent rescue and evidence layer: current state, blockers, risks and the shortest route to demo or launch.

## We already have recruiters

Great. HR Autopilot does not replace recruiters. It gives them a cleaner queue, faster candidate collection, statuses and reporting.

## This is expensive

The diagnostic or pilot is designed to reduce uncertainty before a larger commitment. The cost of another month of unclear work is usually higher than a short evidence-driven sprint.

## Can you guarantee the result?

I do not guarantee magic. I guarantee bounded deliverables: report, blocker map, agreed fixes or funnel setup, evidence and next-step plan.
