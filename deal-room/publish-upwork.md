## Project summary

I help founders and small teams understand why an existing AI-built or contractor-built product is not ready for demo, deployment or launch.

If you already have a repository but the project is hard to run, hard to deploy or hard to verify, I will run a focused Repo Diagnostic and deliver a practical technical state report.

## What makes this different

This is not vague hourly development. It is a bounded diagnostic designed to replace uncertainty with evidence:

- what works;
- what is broken;
- what blocks launch;
- what should be fixed first;
- whether a 7-day rescue sprint is realistic.

## FAQ

### Will you fix the repository in this diagnostic?

The diagnostic is a technical state report. Fixes are handled in a separate rescue sprint after scope is clear.

### Can this help if the project was built with AI tools?

Yes. This offer is specifically designed for AI-built or fast-built MVPs where the code exists but launch readiness is unclear.

### Can you continue after the diagnostic?

Yes. If the diagnostic confirms a practical path, the next step is a 7-day or 14-day rescue sprint.
