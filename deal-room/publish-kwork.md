# Kwork Submission Pack

## Kwork 1 — Repo Diagnostic

### Title

Проведу аудит AI/MVP-репозитория и найду launch blockers

### Category suggestion

Разработка и IT / Доработка сайтов и приложений / Боты и автоматизация / Консультации по IT

### Short description

Проведу техническую диагностику репозитория, который сложно запустить, проверить или подготовить к demo/deploy.

### Full description

Если у вас уже есть код продукта, но проект не запускается, плохо документирован или непонятно готов ли к demo/deploy, я проведу техническую диагностику.

Что входит:

- обзор структуры репозитория;
- список критических блокеров;
- risk map;
- рекомендации по запуску;
- план дальнейшего rescue sprint.

Подходит для:

- AI-generated MVP;
- недоделанных подрядных проектов;
- стартапов перед demo;
- проектов после хаотичной разработки;
- founders, которым нужно понять реальное техническое состояние.

### Package levels

Basic:

- быстрый обзор;
- список проблем;
- рекомендации.

Standard:

- полный отчёт;
- blockers;
- risk map;
- план исправлений.

Premium:

- полный отчёт;
- приоритетный план;
- консультация;
- sprint scope.

### Buyer requirements

1. Ссылка на репозиторий или архив.
2. Описание продукта.
3. Что сейчас не работает.
4. Какой результат нужен.
5. Дедлайн.

## Kwork 2 — HR Autopilot

### Title

Соберу Telegram-воронку для массового подбора кандидатов

### Short description

Настрою структуру Telegram-first воронки для массового подбора: анкета, статусы, передача рекрутеру и отчёт по этапам.

### Full description

Помогаю бизнесу и HR-командам не терять кандидатов между откликом и выходом на смену.

Что входит:

- сценарий Telegram-анкеты;
- вопросы первичного отбора;
- статусы кандидата;
- логика передачи рекрутеру;
- шаблоны сообщений;
- базовый отчёт по воронке.

Подходит для:

- курьеров;
- швей;
- операторов;
- линейного персонала;
- HR-аутсорсинга;
- компаний с постоянным массовым наймом.

### Buyer requirements

1. Вакансия.
2. Город.
3. Условия.
4. Требования.
5. Источники кандидатов.
6. Как сейчас ведёте статусы.
7. Кто отвечает кандидату.

## Kwork response template

Здравствуйте. Спасибо за заявку. Для старта пришлите, пожалуйста, вводные по анкете: цель, текущий процесс, что сейчас не работает и какой результат хотите получить. После этого подтвержу точный scope и сроки.
